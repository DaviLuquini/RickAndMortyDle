RickAndMortyDle, is a Dle game based on the series Rick and Morty, strongly inspired by the famous game Loldle.
The API https://rickandmortyapi.com/
 was consumed to obtain characters and their information.

ðŸ‘‰ Play now: https://rickandmortydle.com/

Note: The backend is fully implemented but currently deactivated.

Built with:

FrontEnd: Angular, HTML, CSS, TypeScript (fully responsive design)
BackEnd: C#, .NET, ASP.NET, PostgreSQL, Docker Containers

Backend design:

Implemented using SOLID principles and Domain-Driven Design (DDD).
Features login and registration functionalities with JWT for session verification.
User data, including profile images and points, is persistently stored.
